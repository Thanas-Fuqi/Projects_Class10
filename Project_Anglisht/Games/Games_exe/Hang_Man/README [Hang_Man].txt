
====================================================================================
Hang_Man version 1.0
====================================================================================

GAME DESCRIPTION
=================
Hang_Man is a minigame in which the goal is to save the hanging stick-man from 
hanging by find the word the computer is "thinking". 
(Note. the word does not contain repeating letters [E.g. swimming two m-s and i-s] )
====================================================================================

GENERAL USAGE NOTES
====================
- Hang_Man is a windows executable it cannot be runed in windows 7 or older versions.
- The screen for optimal usage must be 800px by 600px or bigger.
- The computer must have a keyboard, a mouse and a screen attached. 
====================================================================================

How to run it
==============
You can try to run the program by going firstly in the folder of the cd/dvd drive
and clicking in the Hang_Man shortcut located in Games/Games folder.
If game does not launch go back to the root folder and run the exe file 
located in the Games/Games_code/Hang_Man folder.
====================================================================================

Supported keys and commands
============================
When the game begins you are given the word in which the letters are replaced by "_"
- You can guess only 1 letter at a time. 
- You are given 5 chances and in the 6-th you lose.
- To guess a letter you should type that letter and then press ENTER.
